# 🏦 Bank Matsliakh – Banking Data Analysis Project

## 📌 Project Overview

This project presents an end-to-end banking data analysis solution developed using **SQL, Python, Power BI, and a relational SQL Server database**.

The project analyzes customer banking activity between **2018 and 2023**, transforming raw banking data into meaningful business insights through data modeling, SQL analysis, Python visualizations, and interactive Power BI dashboards.

---

# 🎯 Business Problem

Banks generate large volumes of transactional and customer data every day.

The objective of this project is to analyze customer behavior and banking activity in order to answer key business questions such as:

- Which account types are opened most frequently?
- Which branches perform the best?
- How do deposits change over time?
- When do customers close their accounts?
- Is there a relationship between customer complaints and account closures?

---

# 📊 Business Questions

The analysis answers the following questions:

✅ Number of Savings and Current accounts opened by quarter and year (2018–2022)

✅ Top 5 branches by number of opened accounts

✅ Deposit amounts by quarter and year

✅ Number of closed accounts by month (2023)

✅ Correlation between customer complaints and closed accounts (2023)

---

# 🗄 Database

The project is based on a relational SQL Server database containing six connected tables:

| Table | Description |
|--------|-------------|
| 👨‍💼 Bankers | Bank employees |
| 👥 Clients | Customer information |
| 💳 Accounts | Current & Savings accounts |
| 💰 Deposits | Deposit information |
| 🏠 Loans | Loans and mortgages |
| 💳 Cards | Bank cards |

The database was designed using:

- Primary Keys
- Foreign Keys
- Constraints
- Relational Modeling
- ERD Design

# 📊 Key Insights

- The highest number of new accounts was opened in **2019**.
- The strongest branch performance was achieved by Branches **12** and **92**.
- Deposit activity peaked during **Q4 2022**.
- Account closures increased significantly during **October–December 2023**.
- A **moderate positive correlation (0.435)** was found between customer complaints and account closures.
